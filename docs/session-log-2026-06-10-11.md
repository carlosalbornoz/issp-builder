## 4. Real TOC page numbers (commit `04161a4`)

Two-pass render: pass 1 prints invisible `@@toc:id@@` markers (absolute-positioned,
transparent) at each heading; `scanTocMarkers` (new dep `pdfjs-dist`, server-only,
in `serverExternalPackages`) maps markers to pages from the PDF text layer; pass 2
renders the TOC with real numbers. Gotchas encoded in the implementation:

- Marker scan is case-insensitive (headings use `text-transform: uppercase`, and the PDF
  text layer stores transformed text).
- TOC page-number cell is fixed-width so blank (pass 1) vs digits (pass 2) can't change
  line wrapping/pagination.
